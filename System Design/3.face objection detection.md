# Module 03 — Face & Object Detection

**Role:** Identify people and detect objects in real-time using camera.
**Hardware:** Raspberry Pi 4, Pi Camera Module v2
**Dependencies:** face_recognition, dlib, OpenCV, YOLOv8-nano (Ultralytics)

---

## Tech Stack

| Component | Library | Why |
|---|---|---|
| Face Detection | face_recognition + dlib | Simple API, accurate, runs on Pi |
| Object Detection | YOLOv8-nano | Lightest YOLO variant, Pi-compatible |
| Camera Interface | OpenCV (cv2) | Standard, well-supported |
| Face DB | SQLite + numpy (encodings) | Store face vectors locally |

---

## Architecture

```
[Pi Camera]
  → camera_stream.py         (OpenCV frame capture)
  → face_recognition.py      (detect + identify faces)
      → users DB             (match encoding → name)
      → greeting trigger     (speak name via TTS)
      → attendance_log.py    (timestamp + log)
  → object_detection.py      (YOLOv8 detect objects)
      → action_trigger.py    (notify / log detected objects)
```

### camera_stream.py
```python
import cv2

def get_camera(index: int = 0):
    """Returns OpenCV VideoCapture object."""
    cap = cv2.VideoCapture(index)
    cap.set(cv2.CAP_PROP_FRAME_WIDTH, 640)
    cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 480)
    cap.set(cv2.CAP_PROP_FPS, 10)  # limit FPS to reduce CPU load on Pi
    if not cap.isOpened():
        raise RuntimeError("Camera not found. Check index or connection.")
    return cap

def read_frame(cap):
    """Reads one frame. Returns (success, frame)."""
    return cap.read()

def release_camera(cap):
    cap.release()
    cv2.destroyAllWindows()
```

### face_recognition_module.py
```python
import face_recognition
import numpy as np
import pickle
import os
import sqlite3
from datetime import datetime

ENCODINGS_PATH = "data/known_faces/encodings.pkl"
DB_PATH = "data/robot.db"

def load_encodings() -> tuple:
    """Load precomputed face encodings from disk."""
    if os.path.exists(ENCODINGS_PATH):
        with open(ENCODINGS_PATH, "rb") as f:
            data = pickle.load(f)
        return data["encodings"], data["names"]
    return [], []

def register_face(name: str, image_path: str):
    """
    Register a new person.
    Computes encoding and appends to encodings.pkl
    """
    image = face_recognition.load_image_file(image_path)
    encodings = face_recognition.face_encodings(image)

    if not encodings:
        raise ValueError(f"No face found in {image_path}")

    known_encodings, known_names = load_encodings()
    known_encodings.append(encodings[0])
    known_names.append(name)

    with open(ENCODINGS_PATH, "wb") as f:
        pickle.dump({"encodings": known_encodings, "names": known_names}, f)

    print(f"[Face] Registered: {name}")

def identify_faces(frame) -> list:
    """
    Detects and identifies all faces in a frame.
    Returns list of dicts: [{ name, location }]
    frame: BGR numpy array (from OpenCV)
    """
    known_encodings, known_names = load_encodings()

    # Convert BGR to RGB
    rgb_frame = frame[:, :, ::-1]

    # Detect face locations and encodings
    locations = face_recognition.face_locations(rgb_frame, model="hog")  # use "cnn" for better accuracy but slower
    encodings = face_recognition.face_encodings(rgb_frame, locations)

    results = []
    for encoding, location in zip(encodings, locations):
        matches = face_recognition.compare_faces(known_encodings, encoding, tolerance=0.5)
        name = "Unknown"

        if True in matches:
            distances = face_recognition.face_distance(known_encodings, encoding)
            best_idx = np.argmin(distances)
            if matches[best_idx]:
                name = known_names[best_idx]

        results.append({"name": name, "location": location})

    return results

def log_attendance(name: str):
    """Log face detection to attendance DB."""
    conn = sqlite3.connect(DB_PATH)
    cursor = conn.cursor()
    cursor.execute("""
        INSERT INTO attendance (name, timestamp)
        VALUES (?, ?)
    """, (name, datetime.now()))
    conn.commit()
    conn.close()
```

### object_detection.py
```python
from ultralytics import YOLO
import cv2

# Load model once at startup
model = YOLO("yolov8n.pt")  # nano — lightest model

# Objects we care about in office context
TARGET_OBJECTS = ["cup", "cell phone", "laptop", "person", "chair", "bottle", "book"]

def detect_objects(frame) -> list:
    """
    Runs YOLOv8-nano on a frame.
    Returns list of dicts: [{ label, confidence, bbox }]
    frame: BGR numpy array
    """
    results = model(frame, verbose=False, conf=0.4)  # conf threshold 40%
    detections = []

    for r in results:
        for box in r.boxes:
            label = model.names[int(box.cls)]
            if label in TARGET_OBJECTS:
                detections.append({
                    "label": label,
                    "confidence": round(float(box.conf), 2),
                    "bbox": box.xyxy[0].tolist()  # [x1, y1, x2, y2]
                })

    return detections

def draw_detections(frame, detections: list):
    """Draw bounding boxes on frame (for debugging / web stream)."""
    for d in detections:
        x1, y1, x2, y2 = [int(v) for v in d["bbox"]]
        cv2.rectangle(frame, (x1, y1), (x2, y2), (0, 255, 0), 2)
        cv2.putText(frame, f"{d['label']} {d['confidence']}",
                    (x1, y1 - 8), cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0, 255, 0), 1)
    return frame
```

### vision_loop.py (combined runtime loop)
```python
import cv2
import time
from modules.vision.camera_stream import get_camera, read_frame
from modules.vision.face_recognition_module import identify_faces, log_attendance
from modules.vision.object_detection import detect_objects, draw_detections
from modules.voice.tts import speak

cap = get_camera()
last_greeted = {}  # avoid re-greeting same person every frame
GREET_COOLDOWN = 30  # seconds

print("[Vision] Starting camera loop...")

try:
    while True:
        ret, frame = read_frame(cap)
        if not ret:
            break

        # Face Recognition
        faces = identify_faces(frame)
        for face in faces:
            name = face["name"]
            now = time.time()
            if name != "Unknown":
                if name not in last_greeted or (now - last_greeted[name]) > GREET_COOLDOWN:
                    speak(f"Hello {name}, welcome!")
                    log_attendance(name)
                    last_greeted[name] = now

        # Object Detection (every 5th frame to save CPU)
        objects = detect_objects(frame)

        # Draw for debug (disable in production)
        frame = draw_detections(frame, objects)
        cv2.imshow("Vision", frame)

        if cv2.waitKey(1) & 0xFF == ord('q'):
            break

        time.sleep(0.05)  # ~20 FPS cap

except KeyboardInterrupt:
    pass
finally:
    cap.release()
    cv2.destroyAllWindows()
```

## Hardware Integration

- **Pi Camera:** Enable via `sudo raspi-config` → Interface Options → Camera
- **USB Webcam alternative:** Works with `cv2.VideoCapture(0)` directly
- **Camera position:** Mount at eye level (~1.2–1.5m height) for best face detection angle
- **Lighting:** Low light = poor detection. Add LED ring light near camera for indoor use
- **Frame rate:** Keep at 10 FPS on Pi to avoid thermal throttling

---

## ⚠️ Difficulties

| Issue | Cause | Fix |
|---|---|---|
| dlib compile takes 2+ hours on Pi | ARM compilation | Cross-compile on PC or use prebuilt wheel |
| Face recognition slow per frame | hog model + full frame | Resize frame to 320x240 before processing |
| YOLOv8 first download fails | No internet on Pi | Pre-download `yolov8n.pt` on PC, copy to Pi |
| False face matches | Low tolerance | Lower tolerance from 0.6 → 0.5 or add more photos per person |
| Camera not found | Wrong index | Try `cv2.VideoCapture(0)` then `(1)` |
| Memory error with both YOLO + face_rec | Pi RAM limit | Run on alternating frames (face every 2nd, YOLO every 3rd) |
| Greets same person repeatedly | No cooldown | Implement `last_greeted` dict with timestamp (shown in code) |

## Scalability Notes (Production)

- Replace `face_recognition` + dlib with **InsightFace** or **DeepFace** — faster and more accurate
- Use **Hailo-8 AI accelerator** (HAT for Pi 5) for 10x faster inference
- Replace YOLOv8-nano with **YOLOv8-small** on Jetson for better accuracy
- Add **re-identification (ReID)** model to track people across camera views
- Store face encodings in **vector database (Chroma / Qdrant)** instead of pickle for large scale
