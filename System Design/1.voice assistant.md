# Module 01 — Voice Assistant (Core Brain)

**Role:** Central interaction layer. Listens, understands, responds.
**Hardware:** Raspberry Pi 4, USB Microphone, Speaker
**Dependencies:** Porcupine, Whisper, Ollama (LLaMA), Piper TTS, SQLite

---

## Tech Stack

| Component | Library | Why |
|---|---|---|
| Wake Word | Porcupine (Picovoice) | Lightweight, runs offline on Pi |
| STT | OpenAI Whisper (local) | Accurate, free, no API needed |
| LLM | LLaMA 3.2 3B Q4 via Ollama | Local inference, no cloud cost |
| TTS | Piper TTS | Fast, natural voice, offline |
| Memory | SQLite | Simple persistent storage on Pi |

---

## Architecture

```
[Mic] 
  → wake_word.py        (always listening, low CPU)
  → stt.py              (triggered on wake word)
  → llm_agent.py        (sends prompt + memory context)
  → tts.py              (speaks response)
  → database.py         (saves conversation turn)
```

## Code

### wake_word.py
```python
import pvporcupine
import pyaudio
import struct

def listen_for_wake_word(keyword_path: str, access_key: str, callback):
    """
    Continuously listens for wake word.
    Calls callback() when detected.
    keyword_path: path to .ppn file from Picovoice console
    access_key: free key from picovoice.ai
    """
    porcupine = pvporcupine.create(
        access_key=access_key,
        keyword_paths=[keyword_path]
    )

    pa = pyaudio.PyAudio()
    stream = pa.open(
        rate=porcupine.sample_rate,
        channels=1,
        format=pyaudio.paInt16,
        input=True,
        frames_per_buffer=porcupine.frame_length
    )

    print("[Wake Word] Listening...")

    try:
        while True:
            pcm = stream.read(porcupine.frame_length, exception_on_overflow=False)
            pcm = struct.unpack_from("h" * porcupine.frame_length, pcm)
            result = porcupine.process(pcm)
            if result >= 0:
                print("[Wake Word] Detected!")
                callback()
    except KeyboardInterrupt:
        pass
    finally:
        stream.stop_stream()
        stream.close()
        pa.terminate()
        porcupine.delete()
```

### stt.py
```python
import whisper
import sounddevice as sd
import numpy as np
import scipy.io.wavfile as wav
import tempfile
import os

model = whisper.load_model("base")  # use "tiny" if Pi is slow

def record_audio(duration: int = 5, sample_rate: int = 16000) -> str:
    """Records audio for `duration` seconds. Returns path to temp wav file."""
    print(f"[STT] Recording for {duration}s...")
    audio = sd.rec(
        int(duration * sample_rate),
        samplerate=sample_rate,
        channels=1,
        dtype='int16'
    )
    sd.wait()

    tmp = tempfile.NamedTemporaryFile(suffix=".wav", delete=False)
    wav.write(tmp.name, sample_rate, audio)
    return tmp.name

def transcribe(audio_path: str) -> str:
    """Transcribes audio file to text using Whisper."""
    result = model.transcribe(audio_path)
    os.remove(audio_path)
    return result["text"].strip()

def listen_and_transcribe(duration: int = 5) -> str:
    path = record_audio(duration)
    text = transcribe(path)
    print(f"[STT] Heard: {text}")
    return text
```

### llm_agent.py
```python
import requests
import json
from memory.conversation_memory import get_history, save_turn

OLLAMA_URL = "http://localhost:11434/api/chat"
MODEL = "llama3.2:3b"

SYSTEM_PROMPT = """
You are an AI assistant robot in an office environment.
You help with meetings, answer questions, and assist people.
Be concise, helpful, and friendly.
"""

def ask_llm(user_input: str, user_id: int = 1) -> str:
    """Send prompt to LLaMA with conversation history."""
    history = get_history(user_id, limit=10)

    messages = [{"role": "system", "content": SYSTEM_PROMPT}]
    messages += history
    messages.append({"role": "user", "content": user_input})

    response = requests.post(OLLAMA_URL, json={
        "model": MODEL,
        "messages": messages,
        "stream": False
    })

    reply = response.json()["message"]["content"]
    save_turn(user_id, user_input, reply)
    return reply
```

### tts.py
```python
import subprocess
import tempfile
import os

PIPER_MODEL = "/home/pi/piper/models/en_US-lessac-medium.onnx"

def speak(text: str):
    """Converts text to speech using Piper TTS and plays via speaker."""
    tmp = tempfile.NamedTemporaryFile(suffix=".wav", delete=False)
    tmp.close()

    subprocess.run([
        "piper",
        "--model", PIPER_MODEL,
        "--output_file", tmp.name
    ], input=text.encode(), check=True)

    subprocess.run(["aplay", tmp.name], check=True)
    os.remove(tmp.name)
```

### memory/conversation_memory.py
```python
import sqlite3
from datetime import datetime

DB_PATH = "data/robot.db"

def get_history(user_id: int, limit: int = 10) -> list:
    """Returns last N conversation turns as list of role/content dicts."""
    conn = sqlite3.connect(DB_PATH)
    cursor = conn.cursor()
    cursor.execute("""
        SELECT role, message FROM conversations
        WHERE user_id = ?
        ORDER BY timestamp DESC
        LIMIT ?
    """, (user_id, limit))
    rows = cursor.fetchall()
    conn.close()
    return [{"role": r[0], "content": r[1]} for r in reversed(rows)]

def save_turn(user_id: int, user_msg: str, bot_reply: str):
    """Saves one conversation turn to DB."""
    conn = sqlite3.connect(DB_PATH)
    cursor = conn.cursor()
    now = datetime.now()
    cursor.execute(
        "INSERT INTO conversations (user_id, role, message, timestamp) VALUES (?, ?, ?, ?)",
        (user_id, "user", user_msg, now)
    )
    cursor.execute(
        "INSERT INTO conversations (user_id, role, message, timestamp) VALUES (?, ?, ?, ?)",
        (user_id, "assistant", bot_reply, now)
    )
    conn.commit()
    conn.close()
```

```python
from modules.voice.wake_word import listen_for_wake_word
from modules.voice.stt import listen_and_transcribe

ACCESS_KEY = "YOUR_PICOVOICE_KEY"
KEYWORD_PATH = "config/hey_robot.ppn"

def on_wake_word():
    text = listen_and_transcribe(duration=5)
    if text:
        reply = ask_llm(text)
        speak(reply)

listen_for_wake_word(KEYWORD_PATH, ACCESS_KEY, on_wake_word)
```

## Hardware Integration

- **Microphone:** Connect USB mic to Pi USB port. Check with `arecord -l`
- **Speaker:** 3.5mm jack or USB speaker. Test with `aplay /usr/share/sounds/alsa/Front_Left.wav`
- **Always-on:** Wake word uses very low CPU (~2%) — safe to run 24/7
- **Boot autostart:** Add to `/etc/rc.local` or use `systemd` service

---

## ⚠️ Difficulties

| Issue | Cause | Fix |
|---|---|---|
| LLaMA response too slow (>10s) | Pi 4 limited RAM/CPU | Use Q4 quantized 3B model. Expect 3–8s response |
| Whisper misheard words | Background noise | Use directional USB mic, add VAD (Voice Activity Detection) |
| Piper TTS robotic voice | Default model | Download `en_US-lessac-medium` not `low` quality |
| Wake word false triggers | Sensitivity too high | Tune sensitivity param in Porcupine (0.3–0.5 range) |
| PyAudio install fails on Pi | ALSA dependency | Run `sudo apt install portaudio19-dev` first |
| Ollama not found | PATH issue | Add `export PATH=$PATH:/usr/local/bin` to `.bashrc` |

## Scalability Notes (Production)

- Swap Pi → **Jetson Orin** for faster LLaMA inference (<1s response)
- Swap LLaMA 3B → **LLaMA 3 8B or 70B** for better reasoning
- Add **Redis** instead of SQLite for multi-user session management
- Use **faster-whisper** (CTranslate2) for 4x faster STT on production hardware
- Add **speaker verification** (voiceprint) so robot knows WHO is speaking
