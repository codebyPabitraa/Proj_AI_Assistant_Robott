## Power Management Code

```python
import smbus
import time

class BatteryManagementSystem:
    def __init__(self, i2c_address: int = 0x36):
        """Interface with JK BMS via I2C."""
        self.bus = smbus.SMBus(1)
        self.address = i2c_address

    def read_voltage(self) -> float:
        """Read battery voltage (V)."""
        data = self.bus.read_i2c_block_data(self.address, 0x00, 2)
        voltage = ((data[0] << 8) | data[1]) / 1000.0
        return voltage

    def read_current(self) -> float:
        """Read current (A). Negative = discharge."""
        data = self.bus.read_i2c_block_data(self.address, 0x02, 2)
        current_raw = ((data[0] << 8) | data[1])
        if current_raw > 32767:
            current_raw -= 65536  # Two's complement
        return current_raw / 1000.0

    def read_temperature(self) -> float:
        """Read BMS temperature (°C)."""
        data = self.bus.read_i2c_block_data(self.address, 0x04, 1)
        return data[0]

    def read_capacity_percent(self) -> float:
        """Read state of charge (%)."""
        data = self.bus.read_i2c_block_data(self.address, 0x05, 1)
        return data[0]

    def get_status(self) -> dict:
        """Return complete status."""
        return {
            "voltage_v": self.read_voltage(),
            "current_a": self.read_current(),
            "temp_c": self.read_temperature(),
            "soc_percent": self.read_capacity_percent(),
            "remaining_energy_wh": self.read_voltage() * self.read_capacity_percent() * 100 / 100
        }

    def should_dock(self) -> bool:
        """Return True if battery should be charged soon."""
        status = self.get_status()
        return status["soc_percent"] < 20 or status["temp_c"] > 45

class PowerMonitor:
    def __init__(self, bms: BatteryManagementSystem):
        self.bms = bms
        self.low_voltage_threshold = 42.0  # 48V min safe for Li
        self.critical_threshold = 40.0

    def monitor_loop(self):
        """Run continuously to protect battery."""
        while True:
            status = self.bms.get_status()
            voltage = status["voltage_v"]

            print(f"[Power] {voltage:.1f}V | {status['current_a']:.1f}A | {status['soc_percent']:.0f}%")

            if voltage < self.critical_threshold:
                print("[Power] CRITICAL - Emergency shutdown!")
                # TODO: Trigger graceful shutdown
                break

            if self.bms.should_dock():
                print("[Power] Low battery - returning to dock")
                # TODO: Publish docking signal to navigation

            time.sleep(30)  # Check every 30 seconds
```

## Connector Pinout Reference

```
Anderson Connector (Main Power):
  Red  → +48V
  Black → GND

XT60 Connectors (Motor Circuits):
  + → 12V Motor rail
  - → GND

I2C (LiDAR, BMS, Camera):
  SDA → GPIO 2
  SCL → GPIO 3
  GND → Pin 6

Motor Control (GPIO):
  Motor1 FWD → GPIO 17 (PWM)
  Motor1 REV → GPIO 27
  Motor2 FWD → GPIO 18 (PWM)
  Motor2 REV → GPIO 23

Encoder (GPIO):
  Motor1 Encoder A → GPIO 22
  Motor1 Encoder B → GPIO 24
  Motor2 Encoder A → GPIO 25
  Motor2 Encoder B → GPIO 26

Servo Control (GPIO):
  Gripper Servo → GPIO 12 (PWM)
  Shoulder Joint → GPIO 13 (PWM)
  Elbow Joint → GPIO 16 (PWM)
```

## Assembly Steps

1. **Frame Assembly** (1 hour)
   - Assemble 20x20 aluminum extrusion into rectangle
   - Install motor mounts on each side
   - Mount ball caster at rear

2. **Motor & Drivetrain** (1 hour)
   - Install DC motors on mounts
   - Attach wheels with set screws
   - Connect encoder cables to motor terminal

3. **Electronics Integration** (2 hours)
   - Mount Pi 4 on center plate with thermal standoffs
   - Install BMS on battery compartment
   - Route power cables: battery → BMS → contactor → distribution

4. **Sensor Mounting** (1 hour)
   - LiDAR on front elevated bracket
   - USB camera on arm gripper
   - Mount IMU on frame center

5. **Testing** (1 hour)
   - Continuity check all power circuits
   - Voltage regulation tests (12V, 5V under load)
   - Motor spin test with no load

## ⚠️ Difficulties

| Issue | Cause | Fix |
|---|---|---|
| Motors stall | Insufficient gear ratio | Add 50:1 gearbox |
| Battery drains fast | BMS shuts off early | Recalibrate BMS capacity |
| Unstable Pi boot | Noisy 5V rail | Add bulk capacitor (470µF) |
| Gripper weak grip | Servo underpowered | Upgrade to 6kg servo, mechanical advantage |
| Encoder noise | EMI from motor PWM | Twisted pair shielded cable |
| Extrusion flex | Stress concentration | Add diagonal bracing with cables |
