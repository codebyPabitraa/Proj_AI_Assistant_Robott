# Module 02 — Meeting Assistant

**Role:** Record, transcribe, diarize, summarize meetings and export PDF minutes.
**Hardware:** Raspberry Pi 4, USB Microphone / Mic Array, Speaker
**Dependencies:** Whisper, pyannote.audio, LLaMA (Ollama), reportlab

---

## Tech Stack

| Component | Library | Why |
|---|---|---|
| Real-time Transcription | Whisper (streaming) | Accurate offline STT |
| Speaker Diarization | pyannote.audio | Separates who said what |
| Speaker ID | face_recognition + voice match | Maps voice to known person |
| Summarization | LLaMA 3 via Ollama | Local, no API cost |
| PDF Export | reportlab | Programmatic PDF generation |

---

## Architecture

```
[Mic Array]
  → transcription.py       (Whisper: audio → raw text)
  → diarization.py         (pyannote: split by speaker)
  → speaker_id.py          (match speaker to known user)
  → summarizer.py          (LLaMA: summarize transcript)
  → pdf_export.py          (reportlab: generate PDF)
  → database.py            (save meeting record)
```

## Code

### transcription.py
```python
import whisper
import sounddevice as sd
import numpy as np
import scipy.io.wavfile as wav
import tempfile
import os
from datetime import datetime

model = whisper.load_model("base")

def record_meeting_audio(duration: int = 3600) -> str:
    """
    Records full meeting audio.
    duration: max seconds (default 1 hour)
    Returns path to saved wav file.
    """
    sample_rate = 16000
    print(f"[Meeting] Recording started. Max duration: {duration}s")
    print("[Meeting] Press Ctrl+C to stop early.")

    frames = []
    try:
        with sd.InputStream(samplerate=sample_rate, channels=1, dtype='int16') as stream:
            while True:
                chunk, _ = stream.read(sample_rate)  # 1 second chunks
                frames.append(chunk)
    except KeyboardInterrupt:
        pass

    audio = np.concatenate(frames, axis=0)
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    path = f"data/meeting_records/transcripts/meeting_{timestamp}.wav"
    wav.write(path, sample_rate, audio)
    print(f"[Meeting] Audio saved: {path}")
    return path

def transcribe_audio(audio_path: str) -> str:
    """Transcribes full audio file to text."""
    print("[Meeting] Transcribing...")
    result = model.transcribe(audio_path)
    return result["text"].strip()
```

### diarization.py
```python
from pyannote.audio import Pipeline
import torch

# Load once at startup
pipeline = Pipeline.from_pretrained(
    "pyannote/speaker-diarization-3.1",
    use_auth_token="YOUR_HUGGINGFACE_TOKEN"
)

def diarize(audio_path: str) -> list:
    """
    Returns list of dicts: [{ speaker, start, end }]
    Runs post-meeting (not real-time) to save RAM on Pi.
    """
    print("[Diarization] Processing speakers...")
    diarization = pipeline(audio_path)

    segments = []
    for turn, _, speaker in diarization.itertracks(yield_label=True):
        segments.append({
            "speaker": speaker,
            "start": round(turn.start, 2),
            "end": round(turn.end, 2)
        })

    print(f"[Diarization] Found {len(set(s['speaker'] for s in segments))} speakers")
    return segments

def merge_transcript_with_speakers(transcript: str, segments: list) -> str:
    """
    Merges raw transcript with speaker labels.
    Simple time-based alignment.
    Returns formatted string like:
    SPEAKER_00: Hello everyone...
    SPEAKER_01: Thanks for joining...
    """
    # NOTE: For full word-level alignment, use WhisperX (see difficulty section)
    result = []
    for seg in segments:
        result.append(f"{seg['speaker']} [{seg['start']}s - {seg['end']}s]")
    return "\n".join(result)
```

### summarizer.py
```python
import requests

OLLAMA_URL = "http://localhost:11434/api/chat"
MODEL = "llama3.2:3b"

def summarize_meeting(transcript: str) -> dict:
    """
    Uses LLaMA to generate structured meeting summary.
    Returns dict with summary, action_items, decisions.
    """
    prompt = f"""
You are an expert meeting assistant.
Given the following meeting transcript, extract:
1. A brief summary (3-5 sentences)
2. Action items (bullet list with owner if mentioned)
3. Key decisions made

Transcript:
{transcript}

Respond in this format:
SUMMARY:
<summary here>

ACTION ITEMS:
- <item 1>
- <item 2>

DECISIONS:
- <decision 1>
"""

    response = requests.post(OLLAMA_URL, json={
        "model": MODEL,
        "messages": [{"role": "user", "content": prompt}],
        "stream": False
    })

    content = response.json()["message"]["content"]

    # Parse sections
    result = {"summary": "", "action_items": [], "decisions": []}
    current = None

    for line in content.split("\n"):
        line = line.strip()
        if line.startswith("SUMMARY:"):
            current = "summary"
        elif line.startswith("ACTION ITEMS:"):
            current = "action_items"
        elif line.startswith("DECISIONS:"):
            current = "decisions"
        elif line and current == "summary":
            result["summary"] += line + " "
        elif line.startswith("- ") and current in ("action_items", "decisions"):
            result[current].append(line[2:])

    return result
```

### pdf_export.py
```python
from reportlab.lib.pagesizes import A4
from reportlab.lib.styles import getSampleStyleSheet, ParagraphStyle
from reportlab.platypus import SimpleDocTemplate, Paragraph, Spacer, Table, TableStyle
from reportlab.lib import colors
from reportlab.lib.units import mm
from datetime import datetime
import os

def generate_pdf(
    meeting_title: str,
    transcript: str,
    summary: dict,
    output_dir: str = "data/meeting_records/pdfs/"
) -> str:
    """
    Generates Minutes of Meeting PDF.
    Returns path to generated PDF.
    """
    os.makedirs(output_dir, exist_ok=True)
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    filename = f"{output_dir}MOM_{timestamp}.pdf"

    doc = SimpleDocTemplate(filename, pagesize=A4,
                            rightMargin=20*mm, leftMargin=20*mm,
                            topMargin=20*mm, bottomMargin=20*mm)
    styles = getSampleStyleSheet()
    story = []

    # Title
    story.append(Paragraph(f"Minutes of Meeting", styles["Title"]))
    story.append(Paragraph(meeting_title, styles["Heading2"]))
    story.append(Paragraph(f"Date: {datetime.now().strftime('%d %B %Y, %H:%M')}", styles["Normal"]))
    story.append(Spacer(1, 10*mm))

    # Summary
    story.append(Paragraph("Summary", styles["Heading2"]))
    story.append(Paragraph(summary.get("summary", "N/A"), styles["Normal"]))
    story.append(Spacer(1, 6*mm))

    # Action Items
    story.append(Paragraph("Action Items", styles["Heading2"]))
    for item in summary.get("action_items", []):
        story.append(Paragraph(f"• {item}", styles["Normal"]))
    story.append(Spacer(1, 6*mm))

    # Decisions
    story.append(Paragraph("Key Decisions", styles["Heading2"]))
    for d in summary.get("decisions", []):
        story.append(Paragraph(f"• {d}", styles["Normal"]))
    story.append(Spacer(1, 6*mm))

    # Full Transcript
    story.append(Paragraph("Full Transcript", styles["Heading2"]))
    story.append(Paragraph(transcript[:3000] + "..." if len(transcript) > 3000 else transcript,
                           styles["Normal"]))

    doc.build(story)
    print(f"[PDF] Generated: {filename}")
    return filename
```

### meeting_runner.py (full flow)
```python
from modules.meeting.transcription import record_meeting_audio, transcribe_audio
from modules.meeting.diarization import diarize, merge_transcript_with_speakers
from modules.meeting.summarizer import summarize_meeting
from modules.meeting.pdf_export import generate_pdf

def run_meeting_session(title: str = "Team Meeting"):
    # Step 1: Record
    audio_path = record_meeting_audio()

    # Step 2: Transcribe
    transcript = transcribe_audio(audio_path)

    # Step 3: Diarize (post-meeting)
    segments = diarize(audio_path)
    labeled_transcript = merge_transcript_with_speakers(transcript, segments)

    # Step 4: Summarize
    summary = summarize_meeting(labeled_transcript)

    # Step 5: Export PDF
    pdf_path = generate_pdf(title, labeled_transcript, summary)

    return {
        "transcript": labeled_transcript,
        "summary": summary,
        "pdf_path": pdf_path
    }
```

## Hardware Integration

- **Mic Array (4-channel):** Better speaker separation than single USB mic
- **Placement:** Place robot in center of meeting room for best audio pickup
- **Pi RAM:** pyannote needs ~2GB RAM — close other modules during diarization
- **Storage:** Each 1-hour meeting WAV ≈ 115MB. Keep an external USB drive

---

## ⚠️ Difficulties

| Issue | Cause | Fix |
|---|---|---|
| pyannote uses too much RAM | Full model loaded | Run diarization after meeting ends, not real-time |
| Speaker labels are SPEAKER_00/01 | No name mapping | Cross-reference with face_recognition module (Module 03) |
| Whisper transcript has no timestamps | Default mode | Use `whisper.transcribe(word_timestamps=True)` |
| Transcript + diarization misaligned | Different tools | Use **WhisperX** (combines both — see integration note) |
| LLaMA summary truncates long meetings | Context window limit | Chunk transcript into 2000-word blocks, summarize each |
| pyannote install fails | torch version mismatch | Pin `torch==2.0.1` and install pyannote after |
| HuggingFace token required | pyannote gated model | Create free HF account and accept model license |
